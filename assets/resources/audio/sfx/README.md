# 音效文件占位符

本目录存放游戏的音效文件（SFX）。

## 需要的音效文件

1. **sfx_item_pickup.mp3** - 拿起物品音效
   - 描述：轻柔的摩擦声，提示玩家物品被拿起
   - 时长：约0.2-0.3秒
   - 音调：中低频，柔和
   - 音量：适中

2. **sfx_item_place.mp3** - 物品归位音效（最核心！）
   - 描述：清脆的放置声，给予玩家满足感
   - 时长：约0.3-0.4秒
   - 音调：中高频，清脆
   - 音量：稍大，突出反馈
   - 这是ASMR的核心音效，需要精心设计

3. **sfx_item_wrong.mp3** - 放错位置音效
   - 描述：提示玩家物品放错了位置
   - 时长：约0.2-0.3秒
   - 音调：低频，温和的提示音
   - 音量：较小，不干扰玩家

4. **sfx_item_bounce.mp3** - 弹回音效
   - 描述：物品弹回原位时的弹性音效
   - 时长：约0.3-0.4秒
   - 音调：中频，有弹性感
   - 音量：适中

## 使用说明

1. 在Cocos Creator编辑器中，将实际的音效文件放入此目录
2. 文件名必须与上述名称完全一致
3. 推荐使用MP3格式（兼容性好，体积小）
4. 音效文件应该在AudioManager的preloadSFX()中预加载
5. 音量可以在AudioManager中通过setSFXVolume()调整

## 音效设计原则

- **简洁**：每次交互只有一个音效，避免重叠
- **及时**：音效必须在动作发生时立即播放
- **舒适**：音效要柔和，不宜刺耳
- **清晰**：每种动作的声音特征要明显，玩家能立即识别
