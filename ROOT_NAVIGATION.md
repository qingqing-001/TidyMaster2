# 交付仓库导航说明

当前根目录按交付仓库口径保留导航用途，不直接承载历史 PASS / FAIL / report / setup / note 结论。

如需判断当前仓库对外交付的正式入口与历史资产边界，请按以下顺序阅读：

1. `README.md`
2. `MAIN_LOOP_VERIFICATION.md`
3. `VERIFY_NAMING_HISTORY.md`
4. `roadmap.md`
5. `scripts/README_VERIFY_HISTORY.md`
6. `package.json`

历史显眼资产统一归档于：

- `archive/history/root-signals/`
- `archive/history/internal-scripts/`

当前公开交付入口以顶层文档与 `package.json` 为准。
